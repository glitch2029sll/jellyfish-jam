# jellyfish-jam
Collect treasure as a Jellyfish, stay alive for the best possible score by keeping at least 2 coins on the screen.

## Development Process
This game was made for the Campfire Roanoke 2026 event,this game had to fit the theme of under the surface, which applies because you are underwater. Originally you played as a crab, but a position bug caused for him to be offset on the y axis. Another issue was getting the game to properly end, as the code for when coins get removed was on two seperate things, causing for high values.
